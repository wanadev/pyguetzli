Installation¶

As PyGuetzli is a binding to a C++ library, you will need a recent GCC and GNU Make version to install it. On Debian / Ubuntu, this can be installed with the following command:

sudo apt-get install build-essential

Installing From PYPI¶

To install PyGuetzli from the PYPI package, just run the following command:

pip install pyguetzli

Installing From Sources¶

git clone https://github.com/wanadev/pyguetzli.git
cd pyguetzli
pip install .
